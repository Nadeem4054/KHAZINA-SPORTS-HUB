# 🏏 KHAZINA SPORTS HUB

**Apne Gaon Ka Cricket Hub** - A complete cricket ground management website for village cricket tournaments.

## 📋 Features

- ✅ Team Registration & Management
- ✅ Tournament Creation & Management
- ✅ Match Scheduling
- ✅ Live Score Updates
- ✅ Automatic Leaderboard
- ✅ Admin Dashboard
- ✅ Captain Login
- ✅ Real-time Statistics

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- React Router
- Lucide Icons

**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 🚀 Installation

### Prerequisites
- Node.js (v14+)
- MongoDB
- Git

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

Frontend runs on: `http://localhost:5173`

### Backend Setup

```bash
cd server
npm install
npm run dev
```

Backend runs on: `http://localhost:5000`
